# Bond Viewer

본 프로그램은
한국 회사채(국채 제외) 발행 정보를 조회하는  
개인용 애플리케이션입니다.

data.go.kr 금융위원회 OpenAPI를 사용합니다.

## 실행 환경

- Python 3.10+
- Windows (exe 빌드 지원)

## 파일 항목
bond_viewer.exe
config.example.ini

맨 처음 실행시 반드시 config.ini에 개인 인증키를 입력할 것.

## 개인 인증키 받고 입력하는 법
1. 구글에 "공공데이터포털"를 검색하여 들어가서 회원가입.
2. 구글에 "금융위원회_채권기본정보"를 검색하여 상단에 뜬 검색 결과로 들어가기.
3. 우측상단의 "활용신청"을 클릭하여 정보를 입력하고 신청하기.(자동 승인이라 아무렇게나 써도 되긴 함.)
4. 마이페이지에 들어가 인증키(영문숫자로 구성된 64글자)를 복사.
5. 압축파일에 동봉된 config.example.ini을 메모장으로 열어 "service_key="의 다음 글자를 전부 삭제한 후
   복사한 인증키를 붙여넣고 저장하고 닫기.
6. config.example.ini의 파일명을 config.ini로 변환.
6. bond_viewer.exe 실행.

※개인 인증키는 2년마다 갱신 필요.


## 프로그램 사용법
첫 실행시 API로드로 인해 시간이 걸림. 약 1~2분. 약 262페이지 로드.
두번째 실행부터 캐시데이터(bond_cache.json)로 먼저 표시 후 백그라운드로  API로드. 로드가 끝나면 새로운 정보로 치환.
그러니 왠만하면 bond_cache.json는 삭제하지 말 것.
config.ini는 절대 삭제하지 말 것.

기업명 / ISIN명 검색 -> 기업명이나 ISIN코드명으로 검색 가능.
공모만 보기 -> 체크를 하면 공모채만 표시.
금리 높은순 -> 현재 화면에서 금리가 높은순으로 정렬.
발행일 최신 순 -> 현재 화면에서 발행일자가 제일 최근인 순으로 정렬.
초기화 -> 검색 결과(검색명, 금리 높은 순, 발행일 최신 순 상태)를 초기화.
새로고침(API) -> 새로운 정보로 갱신.
CSV 내보내기-> 현재 화면에 보이는 결과를 CSV로 저장.

각 항목 우클릭시  ISIN코드 복사 가능 (복수선택 복사 가능)

프로그램 종료시  트레이아이콘으로 전환. 
완전 종료를 원할 시 윈도우 화면 오른쪽 아래의 트레이아이콘(회사채 조회)를 우클릭 후 종료.
